## EMV Workshop

### Configure the Simulated smart card
```bash
cd jcardsim
mvn clean install
cd target
cp ../jcardsim.cfg .
```

#### Hello World applet
Content of `jcardsim.cfg` should be:
```
com.licel.jcardsim.card.applet.0.AID=010203040506070809
com.licel.jcardsim.card.applet.0.Class=com.licel.jcardsim.samples.HelloWorldApplet
com.licel.jcardsim.terminal.type=2
com.licel.jcardsim.vsmartcard.host=localhost
com.licel.jcardsim.vsmartcard.port=35963
#com.licel.jcardsim.terminal.host=localhost
#com.licel.jcardsim.terminal.port=1234
```

Interact with `HelloWorldApplet` application:
```bash
opensc-tool --reader 0 \
--send-apdu 80b800001009010203040506070809050000020F0F7f \
--send-apdu 00a4040009010203040506070809 \
--send-apdu 0102000000
```

Here we have the HelloWorldApplet working.

#### Simulated financial applet

Add `emv-card-simulator` to classpath:

````bash
java -cp "jcardsim-3.0.5-SNAPSHOT.jar:/path/to/emv-card-simulator/build/libs/emv-card-simulator.jar" com.licel.jcardsim.remote.VSmartCard jcardsim.cfg
````

Content of `jcardsim.cfg` should be:
```
com.licel.jcardsim.card.applet.0.AID=010203040506070809
com.licel.jcardsim.card.applet.0.Class=emvcardsimulator.PaymentApplication
com.licel.jcardsim.terminal.type=2
com.licel.jcardsim.vsmartcard.host=localhost
com.licel.jcardsim.vsmartcard.port=35963
#com.licel.jcardsim.terminal.host=localhost
#com.licel.jcardsim.terminal.port=1234
```

Test the applet:

```bash
opensc-tool --reader 0 --send-apdu 80b800001009010203040506070809050000020F0F7f
```
